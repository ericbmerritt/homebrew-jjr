# homebrew-jjr

Homebrew tap for [jjr](https://github.com/ericbmerritt/jujutsu-review) — local
terminal review surface for `jj` stacks generated by an agent.

## Install

```bash
brew tap ericbmerritt/jjr
brew install jjr
```

Or, until the first tagged release lands, build from `main`:

```bash
brew install --HEAD ericbmerritt/jjr/jjr
```

`jjr` requires `jj` (jujutsu) at runtime. Homebrew installs it as a dependency.
The `claude` CLI is optional — only needed for `jjr claude` and the in-TUI `C`
keybind.

## Updating the formula

The
[`release-brew.yml`](https://github.com/ericbmerritt/jujutsu-review/blob/main/.github/workflows/release-brew.yml)
workflow in the source repo opens a PR here on every tag push, updating
`Formula/jjr.rb`'s `url` + `sha256`. Just merge the PR.

The very first formula (current state) needs a manual `sha256` fill-in once
`v0.1.0` is tagged in the source repo:

```bash
curl -sL https://github.com/ericbmerritt/jujutsu-review/archive/refs/tags/v0.1.0.tar.gz | shasum -a 256
```

From `v0.2.0` onward this is automated.
